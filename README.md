# Mart_analysis
mart data analysis using python and sql

## Project OverView
This project analyzes walmart sales data to find useful business insights. It uses Python to clean and analyze data, SQL to query and explore information, and data analysis techniques to answer important business questions. It is a good project for learning data analysis, data handling, and SQL skills.

## Steps involved:
Set up the project environment in VS Code.
Download Walmart sales data from Kaggle using the Kaggle API.
Load the data into Python using Pandas.
Explore the dataset to understand its structure and identify issues.
Clean the data by removing duplicates, handling missing values, and fixing data types.
Create new features, such as calculating the total sales amount for each transaction.
Store the cleaned data in MySQL and PostgreSQL databases.
Use SQL queries to answer business questions.

## Outcome:
The project helps uncover valuable business insights from Walmart sales data while developing practical skills in Python, SQL, data cleaning, database management, and data analysis.
