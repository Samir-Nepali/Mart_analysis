# 🛒 Walmart Sales Data Analysis Project

## 📌 Project Overview

This project is an end-to-end data analysis solution designed to extract meaningful business insights from Walmart sales data. It uses **Python for data processing**, **SQL for database querying**, and structured data analysis techniques to answer key business questions.

The project helps develop practical skills in **data cleaning, data manipulation, SQL querying, and database management**.

---

## 🧰 Tools & Technologies Used

* Python (Pandas, SQLAlchemy)
* SQL (PostgreSQL)
* Kaggle 
* Jupyter Notebook 
---

## 📂 Project Workflow

### 1. Environment Setup

* Configure **Jupyter Notebook**
* Set up Python environment

---

### 2. Kaggle 

* Download  walmart sales dataset from Kaggle 

---

### 3. Exploratory Data Analysis (EDA)

* Use `.head()`, `.info()`, `.describe()` to understand data
* Identify missing values and data types
* Understand data distribution and structure

---

### 4. Data Cleaning

* Remove duplicate records
* Handle missing values
* Fix incorrect data types (dates, numbers, etc.)
* Standardize currency and numerical formats
* Validate cleaned dataset

---

### 5. Feature Engineering

* Create new column: **Total Amount = Unit Price × Quantity**
* Enhance dataset for better analysis and SQL queries

---

### 6. Load Data into Databases

* Connect Python to **PostgreSQL** using SQLAlchemy
* Create database tables automatically
* Insert cleaned data into both databases
* Verify data insertion using SQL queries

---

### 7. SQL Analysis & Business Insights

Perform advanced SQL queries to solve business problems:

* Revenue trends by branch and category
* Best-selling product categories
* Sales performance by city and time
* Customer payment method analysis
* Peak sales time identification
* Profitability by branch and category

---

### 8. Project Documentation & Publishing

* Document all steps in a **README**
* Include SQL queries and analysis results
* Upload project to **GitHub**
